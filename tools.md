# Tools Overview

This project currently provides a collection of free, client‑side utilities for working with PDFs, images, DOCX files, and miscellaneous data formats. All tools run in the browser, ensuring privacy and fast results.

## Current Tools

### PDF Tools
- [x] PDF to Base64
- [x] PDF Inspector
- [x] PDF Renamer
- [x] PDF Viewer
- [x] PDF Merger
- [x] PDF Page Counter
- [x] PDF to Text
- [x] PDF to Image
- [x] Image to PDF
